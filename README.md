# 🖥️ JUANJOSE - Reparación PC & Laptops

Sitio web profesional para servicios de reparación de computadoras y laptops. Desarrollado con tecnologías modernas y diseño responsive.

## 🌐 URL del Sitio
**GitHub Pages:** https://teloinstalo.github.io/juanjose-servicios-web/

## 📱 Características del Sitio

### 🏗️ **Estructura del Proyecto**
- **6 páginas HTML completas** con navegación responsiva
- **Formulario de contacto** con 12 tipos de campos y validación JavaScript
- **Diseño responsive** con Flexbox y CSS Grid
- **SVG favicon** personalizado y wave sections con clip-path
- **Animaciones** W3Schools y efectos hover
- **Información de contacto** actualizada para 2026

### 🎨 **Tecnologías Utilizadas**
- **HTML5** - Estructura semántica
- **CSS3** - Variables CSS, Flexbox, Grid, Animaciones
- **JavaScript** - Validación de formularios, interacciones
- **Google Fonts** - Inter + Poppins para tipografía jerárquica
- **SVG** - Iconos y favicon vectoriales

### 📂 **Páginas del Sitio**
1. **🏠 Inicio** - Página principal con presentación y servicios destacados
2. **🛠️ Servicios** - Catálogo completo de servicios de reparación
3. **📞 Contacto** - Formulario de contacto con validación avanzada
4. **💰 Presupuesto** - Calculadora interactiva de presupuestos
5. **👤 Sobre Mí** - Información personal y experiencia
6. **🔄 Proceso** - Metodología de trabajo paso a paso

### 📱 **Características Técnicas**
- ✅ **Responsive Design** - Mobile, Tablet, Desktop
- ✅ **Optimización SEO** - Meta tags, títulos, descripciones
- ✅ **Accesibilidad** - ARIA labels, navegación por teclado
- ✅ **Performance** - Optimización de imágenes, carga rápida
- ✅ **Cross-browser** - Compatible con Chrome, Firefox, Safari

## 📋 **Formulario de Contacto**

El formulario incluye 12 tipos de campos diferentes:
1. **Texto** - Nombre completo
2. **Email** - Con validación regex
3. **Teléfono** - Formato internacional
4. **Fecha** - Selector de fecha
5. **Select** - Tipo de servicio
6. **Radio buttons** - Urgencia
7. **Checkboxes** - Servicios adicionales
8. **Textarea** - Mensaje detallado
9. **File upload** - Imágenes/documentos
10. **Range** - Nivel de urgencia
11. **Color** - Preferencia de tema
12. **Submit** - Validación antes de enviar
